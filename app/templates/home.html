LiteParse Resume Parser API

Extract structured information from resumes (PDF/DOCX) quickly and reliably.

Quick Start

  1. Step 1 Register to get your API key.
  2. Step 2 Login to get JWT access token.
  3. Step 3 Use the /parse endpoint with JWT or API key.
